## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/ai-powered-symptoms-checker.git
```

Move into the project

```bash
cd ai-powered-symptoms-checker
```

Install dependencies

```bash
npm install
```

Create an environment file

```env
GEMINI_API_KEY=your_gemini_api_key
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

## 🧠 How It Works

1. User enters symptoms.
2. The application sends the symptoms to the **Gemini 2.5 Flash API**.
3. Gemini analyzes the symptoms using AI.
4. The application displays:
   - Possible conditions
   - General recommendations
   - Precautions
   - Next suggested steps
5. The consultation is saved in the patient's history for future reference.

---

## 📖 Patient History

The application stores previous symptom analyses, allowing users to:

- View previous consultations
- Compare symptoms over time
- Revisit AI-generated recommendations
- Maintain a personal health timeline
